# funkcja

Log function body and name to browser's console.

```JS
funkcja(mySuperFunction)
```

## installation
```
npm i funkcja
```

or

```
yarn install funkcja
```

## import
```JS
import { funkcja } from 'funkcja';
```

or

```JS
const { funkcja } = require("funkcja");
```
